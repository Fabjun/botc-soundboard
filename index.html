ABOUT

This tool was made by me for the purpose of supporting my adventures as a game master — and maybe it will help you too.

The app is designed to work in the simplest way possible without limiting your creative freedom. Its one and only purpose is interactive sound design managed live by a game master. This is not a professional sound editing tool.

WHAT IS A PAD?

A PAD is a programmable button you can use to play any sound you like. By this design you can trigger sounds from any device with any interface — a phone or a computer, a keyboard or a numpad, a touchscreen or a mouse.

PAD — Professional Atmospheric Delusion

SINGLE PADS

Each Single-PAD has one of three behaviours:

SOLO
A single sound file. Press the pad and the sound plays once.
LOOP
A single sound file with a configurable repeat count — or set to ∞ for an endless background loop.
LIST
A playlist of multiple sound files played in sequence or shuffled. The whole list can be looped.
COMBO-PADS

A Combo-PAD combines multiple Single-PADs into one programmable sequence. Inside the Combo editor you arrange PADs into Steps.

PADs within the same Step play simultaneously.
Steps run one after another — the next step begins only when all sounds of the previous one have finished.
An empty Step acts as a timed pause — set its length in seconds.
A LOOP set to ∞ inside a Step runs as a background track — it does not block the next Step and stops automatically when the combo ends.

Combo-PADs cannot be nested inside another Combo-PAD.

TWO MODES

The app has two modes switchable via the SETUP | PLAY toggle at the top of the board. In SETUP mode you design and configure your pads. In PLAY mode every key press or tap triggers a sound — nothing can be accidentally edited.

Made with obsessive attention to game night atmosphere.

TIPS & TRICKS

SETUP|PLAY
SETUP ↔ PLAY MODE
The SETUP|PLAY switch at the top of the board toggles between modes. In Setup mode (blue) you can edit pads. In Play mode (gold) keys and taps trigger sounds.
↕ Drag
MOVING PADS
In Setup mode, hold a pad and drag it to another position — the two pads swap places. A short tap opens the edit panel. Drag & drop is disabled in Play mode.
Fade
FADE-IN / FADE-OUT
Configurable per pad in the edit panel (0–10s). Fade-In gradually raises the volume when the sound starts. Fade-Out lowers it when stopping. Enter stops immediately without a fade.
Space
PAUSE / RESUME
Pauses all running sounds and resumes them on the next press. Works on every screen and in Play mode. Exception: when a text input field is active.
Enter
STOP ALL
Immediately stops all running sounds and resets their position to the beginning. Works on every screen and in Play mode.
Escape
BACK / CANCEL
Navigates back to the main menu. If the edit panel is open, Escape closes it (= Cancel). Has no effect in Play mode.
Tap
ASSIGN SOUND-PAD
In Setup mode, tap a pad to open the edit panel. Click the key field and press the desired key — the code is detected automatically.
KEY CONFLICT
When saving, the app checks whether a key is already assigned to another pad. If there is a conflict, a warning appears with the options Cancel or Overwrite.
1 ≠ 1
NUMPAD VS. NUMBER KEY
The "1" on the numpad and the "1" on the main keyboard are detected as different codes internally. Assigning one never accidentally affects the other.
⊞ LIB
THE AUDIO LIBRARY
Every audio file you upload is stored once in the global Library and identified by a fingerprint. Uploading the same file twice is detected automatically — no duplicates. Open the Library from the main menu to browse, upload, or delete files and see which ones are in use.
FROM LIB
REUSE AUDIO WITHOUT RE-UPLOADING
In the pad edit panel, tap "FROM LIB" next to the file picker to assign audio already in the Library — no second copy is stored. Multiple pads can share the same file. Great for recurring sounds like dice rolls or ambient loops.
C-PAD
COMBO-PAD: STEPS IN SEQUENCE
A Combo-PAD plays a sequence of steps one after another. Each step can contain one or more pads that play simultaneously. Tap in Setup mode to build one. Combo-PADs can be nested inside other Combos — but a Combo cannot reference itself or create a loop.
Pause
TIMED PAUSE IN A COMBO
Add a step with no pads and set a duration to create a silent pause between two steps. Useful for dramatic timing — e.g. a quiet moment before a thunder clap. Duration is set per step in the Combo editor.
∞ BG
BACKGROUND TRACKS IN COMBOS
A pad set to ∞ LOOP inside a Combo step runs as a non-blocking background track — the sequence advances to the next step immediately without waiting for it to finish. Use it for ambient music that plays under a sequence of sound effects.
⬇ EXP
EXPORT & IMPORT YOUR LIBRARY
Settings → DATA → EXPORT LIBRARY saves everything — all boards, pads, audio files and settings — into one compressed file. IMPORT LIBRARY merges a backup into your current data: you choose per audio file and per board what to keep, replace or skip. Use this to transfer your setup to another device or browser.
⚙ BAK
PROTECTING YOUR DATA
Browser storage can be cleared unexpectedly. On desktop: enable AUTO-BACKUP in Settings → DATA to save 3 rotating backup files to a local folder every 10 minutes (requires browser support). On iPhone or if auto-backup is unavailable: export manually after every session and save the file to iCloud or Files. The export age is shown in Settings so you always know how fresh your last backup is.
⊞ BRD
MULTIPLE BOARDS
Create as many boards as you like from MY BOARDS in the main menu. Each board has its own set of pads but shares the same audio Library. Rename boards with the ✎ button. All boards are included in every export.
